# ポケスリ画像結合

ポケモンスリープのポケモンステータス画像を、ブラウザ内でグリッド結合するHTMLアプリです。

画像は端末内のブラウザで処理されます。アップロード処理や外部通信はありません。

## できること

- 画像のファイル選択
- ドラッグ＆ドロップでの画像追加
- サンプル画像の読み込み
- 行・列の指定
- 画像の上下左右移動
- 画像の削除
- 削除後の空きマス保持
- 空きマスを白背景として出力
- プレビュー表示
- プレビュー上の座標表示ON/OFF
- レイアウトお気に入り3枠
- PNG保存
- 元画像の解像度を維持した結合

## 使い方

1. `画像を選択` またはドラッグ＆ドロップで画像を追加します。
2. `行` と `列` を指定します。
3. 必要に応じて画像を上下左右に移動します。
4. 不要な画像は `削除` します。削除した位置は空きマスとして残ります。
5. プレビューを確認します。
6. 必要に応じて `座標表示` をONにします。
7. `PNG保存` で結合画像を保存します。

保存ファイル名は次の形式です。

```text
pokemonsleep-img-merge-YYYYMMDD-HHMMSS.png
```

## 補足

- `座標表示` はプレビュー確認用です。保存PNGには座標ラベルを入れません。
- レイアウトお気に入りはブラウザのlocalStorageに保存されます。
- 画像データ本体はlocalStorageに保存しません。
